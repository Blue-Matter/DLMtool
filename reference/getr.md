# Internal function to estimate r

Internal function to estimate r

## Usage

``` r
getr(x, Data, Mvec, Kvec, Linfvec, t0vec, hvec, maxage, r_reps = 100)
```

## Arguments

- x:

  iteration number

- Data:

  Object of class `Data`

- Mvec:

  M

- Kvec:

  K

- Linfvec:

  Linf

- t0vec:

  t0

- hvec:

  steepness

- maxage:

  maximum age

- r_reps:

  number of reps
