# Incremental Index Target MP Internal Function - Effort MP

Incremental Index Target MP Internal Function - Effort MP

## Usage

``` r
Itargeteff_(x, Data, reps, plot, yrsmth, Imulti)
```

## Arguments

- x:

  Iteration number

- Data:

  An object of class `Data`

- reps:

  Number of replicates

- plot:

  Logical. Show the plot?

- yrsmth:

  Years over which to calculate index

- Imulti:

  Parameter controlling how much larger target CPUE / index is compared
  with recent levels.

## Value

A named list with Effort recommendations
